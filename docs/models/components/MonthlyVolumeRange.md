# MonthlyVolumeRange

The low value in each range is included. The high value in each range is excluded.


## Values

| Name                         | Value                        |
| ---------------------------- | ---------------------------- |
| `UNDER10K`                   | under-10k                    |
| `TENK50K`                    | 10k-50k                      |
| `FIFTYK100K`                 | 50k-100k                     |
| `ONE_HUNDREDK250K`           | 100k-250k                    |
| `TWO_HUNDRED_AND_FIFTYK500K` | 250k-500k                    |
| `FIVE_HUNDREDK1M`            | 500k-1m                      |
| `ONEM5M`                     | 1m-5m                        |
| `OVER5M`                     | over-5m                      |
