# FeeModel

Specifies the pricing model used for the calculation of the final fee.

## Example Usage

```java
import io.moov.sdk.models.components.FeeModel;

FeeModel value = FeeModel.FIXED;
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `FIXED`    | fixed      |
| `BLENDED`  | blended    |
| `VARIABLE` | variable   |
