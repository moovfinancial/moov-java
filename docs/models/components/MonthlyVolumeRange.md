# MonthlyVolumeRange

The low value in each range is included. The high value in each range is excluded.


## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `UNDER10K`                     | under-10k                      |
| `TEN_K50_K`                    | 10K-50K                        |
| `FIFTY_K100_K`                 | 50K-100K                       |
| `ONE_HUNDRED_K250_K`           | 100K-250K                      |
| `TWO_HUNDRED_AND_FIFTY_K500_K` | 250K-500K                      |
| `FIVE_HUNDRED_K1_M`            | 500K-1M                        |
| `ONE_M5_M`                     | 1M-5M                          |
| `OVER5M`                       | over-5m                        |
