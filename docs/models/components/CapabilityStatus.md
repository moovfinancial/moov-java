# CapabilityStatus

The status of the capability requested for an account.

## Example Usage

```java
import io.moov.sdk.models.components.CapabilityStatus;

CapabilityStatus value = CapabilityStatus.ENABLED;
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ENABLED`   | enabled     |
| `DISABLED`  | disabled    |
| `PENDING`   | pending     |
| `IN_REVIEW` | in-review   |
